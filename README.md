# 玄女角色解析 - 造梦无双

> 造梦无双玄女角色的深度解析网站，涵盖技能数值、机制分析、觉醒选择与玩法推荐。

## 项目简介

本项目将 4399 论坛帖子 [【结弦】玄女角色解析](https://bbs.4399.cn/thread-tid-51982611) 的内容重新设计为独立的静态网站，提供清晰的阅读体验和完善的交互功能。

**主要功能：**

- 章节式内容展示（基础说明、常态技能组、无双形态、被动技能、设计评析、玩法推荐）
- 白天 / 夜间主题切换（默认跟随系统）
- 桌面端固定目录导航 + 移动端折叠菜单
- 图片懒加载与平滑滚动
- 全设备响应式适配

## 技术栈

| 类别     | 技术                                       |
| -------- | ------------------------------------------ |
| 构建工具 | [Vite](https://vite.dev/) v8               |
| 前端     | 原生 HTML / CSS / JavaScript（无框架依赖） |


## 快速开始

### 安装依赖

```bash
npm install
```

### 本地开发

```bash
npm run dev
```

启动 Vite 开发服务器，支持热更新。浏览器打开终端提示的地址即可访问。

### 构建生产版本

```bash
npm run build
```

输出目录为 `dist/`，所有静态资源会被打包并添加内容哈希以优化缓存。

### 预览构建产物

```bash
npm run preview
```

在本地启动静态服务器预览 `dist/` 目录内容。

## 主题系统

- **白天模式**：浅色背景，默认启用
- **夜间模式**：深色背景
- **跟随系统**：首次访问时根据 `prefers-color-scheme` 自动选择
- 用户手动切换后会记住偏好（存储于 `localStorage`）

主题基于 CSS 自定义属性（CSS Variables）实现，所有颜色变量定义在 `styles/main.css` 中。
