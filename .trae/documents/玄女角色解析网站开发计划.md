# 造梦无双角色玄女解析网站开发计划

## 一、项目概述

复刻 4399 论坛帖子 `https://bbs.4399.cn/thread-tid-51982611` 的玄女角色解析内容，重新设计样式和布局。

### 核心要求

* 内容保持一致，仅改变样式和布局

* 内容居中显示，左右留有适当边距

* 页面右侧固定目录导航

* 资源懒加载提升性能

* 移动端适配

* 白天/夜间主题切换（默认跟随系统）

* 代码需有注释

***

## 二、技术方案

### 2.1 技术选型

* **原生 HTML + CSS + JavaScript**：轻量级、无依赖、加载快

* **无需框架**：性能最优，维护简单

### 2.2 文件结构

```
d:\Desktop\code\test1\
├── index.html          # 主页面
├── styles/
│   └── main.css        # 样式文件
├── scripts/
│   └── main.js         # 脚本文件
└── assets/             # 静态资源目录（用于本地图片）
```

### 2.3 主题系统实现

* 使用 CSS 变量 (`--bg-color`, `--text-color`, 等)

* 通过 `prefers-color-scheme` 检测系统主题

* 提供手动切换按钮覆盖系统设置

* 主题偏好存储于 localStorage

### 2.4 响应式布局

* 桌面端：左侧内容区（最大宽度 800px 居中）+ 右侧固定目录（280px）

* 平板/移动端：单栏布局，目录变为顶部可折叠导航

* 断点设计：768px（平板）、480px（手机）

### 2.5 懒加载策略

* 图片使用 `loading="lazy"` 属性

* 目录导航使用 Intersection Observer 实现滚动高亮

* 非首屏内容延迟渲染

## 三、页面结构

### 3.1 整体布局

```
┌─────────────────────────────────────────────────────────┐
│                      顶部导航栏                          │
├───────────────────────────────────┬─────────────────────┤
│                                   │                     │
│                                   │    固定右侧目录      │
│          主内容区域                │    (滚动时固定)      │
│          (居中, max-800px)        │                     │
│                                   │                     │
│                                   │                     │
└───────────────────────────────────┴─────────────────────┘
```

### 3.2 HTML 章节结构

按照原帖分为六章：

* **第0章**：目录/导引

* **第1章**：基础说明

* **第2章**：常态技能组

* **第3章**：无双形态技能组

* **第4章**：被动技能

* **第5章**：角色设计思路评析

* **第6章**：玩法推荐与性能评价

每章使用 `<section>` 标签，配合 `id` 便于目录跳转。

### 3.3 目录导航

* 固定在右侧（桌面端）

* 实时反映当前阅读位置（滚动高亮）

* 点击跳转对应章节

* 移动端：顶部汉堡菜单，点击展开

## 四、样式设计

### 4.1 配色方案

**白天主题**

| 变量               | 色值      | 用途     |
| ---------------- | ------- | ------ |
| --bg-primary     | #ffffff | 主背景    |
| --bg-secondary   | #f5f5f7 | 次级背景   |
| --text-primary   | #1d1d1f | 主文字    |
| --text-secondary | #6e6e73 | 次级文字   |
| --accent         | #0066cc | 强调色/链接 |
| --border         | #d2d2d7 | 边框     |

**夜间主题**

| 变量               | 色值      | 用途     |
| ---------------- | ------- | ------ |
| --bg-primary     | #1c1c1e | 主背景    |
| --bg-secondary   | #2c2c2e | 次级背景   |
| --text-primary   | #f5f5f7 | 主文字    |
| --text-secondary | #98989d | 次级文字   |
| --accent         | #0a84ff | 强调色/链接 |
| --border         | #38383a | 边框     |

### 4.2 字体

* 标题：`"PingFang SC", "Microsoft YaHei", sans-serif`

* 正文：`"PingFang SC", "Microsoft YaHei", sans-serif`

* 代码/数据：使用等宽字体展示

### 4.3 间距系统

* 基础单位：8px

* 章节间距：48px

* 段落间距：16px

* 表格内边距：12px
